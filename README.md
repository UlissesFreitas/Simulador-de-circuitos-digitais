# Simulador-de-circuitos-digitais
Simulador  de circuitos digitais

Projeto da disciplina de programação avançada

O objetivo é desenvolver em C++ um programa 
simulador de circuitos lógicos, composto por
portas lógicas de 2 ou mais entradas (ou de 
uma entrada, no caso da NOT) dos seguintes
tipos:
NOT (NEGAÇÃO)
AND (E), NAND (NOT AND)
OR (OU), NOR (NOT OR)
XOR (OU EXCLUSIVO), NXOR (NOT XOR)
